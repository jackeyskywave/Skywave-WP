== Description ==

**Wholesale for WooCommerce Lite** is a WooCommerce extension that gives you the ability to display wholesale prices on your WooCommerce store products. Why create a new website to sell your wholesale products when you can use Wholesale for WooCommerce to sell both retail and wholesale products on the same site.

**You can sign-up the customer via a wholesale registration form and then assign them a “Wholesaler” role – it is that simple.**

Apart from adding wholesale pricing to your existing products, you can also add wholesale prices to each variation of a variable product. 

**What’s more exciting for your customers to see they have saved big on a discount?** Display the amount they’re saving on your products and create a big-save buzz.

== Wholesale for WooCommerce Lite Features: ==

* Add wholesale prices to your products
* Wholesale prices are only visible to Wholesaler role customers
* Add wholesale prices to product variations
* Display discount amount (save price) with product price
* Users can submit their request for wholesaler role
* Enable/disable auto-approval of user requests
* Hide retail prices from products 
* Add percentage amount on wholesale products
* Add fixed amounts on wholesale products
* Translation using .PO .MO files
* Ability to change the text on the frontend labels


== Wholesale for WooCommerce Pro ==

Control everything you want to sell to your wholesale customers through [Wholesale for WooCommerce Pro](https://woocommerce.com/products/wholesale-for-woocommerce/?aff=2878) that lets you manage your eCommerce store’s **wholesale pricing, multiple wholesale user roles, and much more**.

Simply sign-up your wholesale customers through a **wholesale registration form** and manage **approval requests** manually or automatically.

Through the **Multiple User Role** creation feature, you can define different discounts for each role with support for both simple and variable products. The **Single User Role** allows you to create and designate a single user role so that they see the wholesale prices as per their given role.

== Installation ==
To add a WordPress Plugin using the built-in plugin installer:

Go to Plugins > Add New.

1. Type in the name \"WooCommerce Wholesale Pricing\" in Search Plugins box
2. Find the \"WooCommerce Wholesale Pricing\" Plugin you wish to install.
3. Click Install Now to install the WordPress Plugin.
4. The resulting installation screen will list the installation as successful or note any problems during the install.
If successful, click Activate Plugin to activate it, or return to plugin installer for further actions.
